Abstract

This ontology is a profile of several ontologies implemented to govern Linked Data resources published within the LOC-I project. It profiles the GeoSPARQL and DCAT v2 & VOID ontologies and requires that certain properties defined by them be present for several classes of resource that are of particular importance to the LOC-I project.

This ontology contains normative constrains on what constitutes LOC-I objects and supplies executable constraints to validate instances. This ontology's components and its relations to the ontologies it profiles are described using the Profiles Vocabulary (PROF).
